# Rio Children's Hospital

Next.js App Router site for Rio Children's Hospital.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Structure

```txt
src/
  app/
    layout.jsx
    page.jsx
    globals.css
  components/
    *.jsx
  data/
    siteData.js
public/
  assets/
```

The existing UI sections, assets, data, and styling are organized for the Next.js App Router.
